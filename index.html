Sobre o Projeto

Este é um aplicativo web simples para registrar e gerenciar tarefas. Ele foi desenvolvido com HTML, CSS, JavaScript e Bootstrap, com foco em praticidade e aprendizado.

  • - Adicione tarefas rapidamente
  • - Remova com um clique
  • - Dados salvos no navegador

Explore a aba "Tarefas" para começar a usar

Dicas de uso

  • Use o menu acima para navegar
  • Na aba "Tarefas", digite e cadastre
  • Clique em uma tarefa para removê-la
  • Os dados ficam salvos mesmo após fechar o navegador
